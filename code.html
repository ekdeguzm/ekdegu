Dryclo

Dryclo informs users of incoming rain via text to prevent hanging clothes from becoming soaked. The program leverages the OpenWeatherMap API to retrieve weather information and then sends a text message using the Twilio service API to notify the user about incoming weather conditions: type, description, temperature, humidity, wind speed and rain depth.
